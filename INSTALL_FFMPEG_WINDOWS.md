# FFmpeg Installation Guide for Windows

## РюЁ рЃЏрЃћрЃЌрЃЮрЃЊрЃў 1: winget (рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў - рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃўрЃарЃћрЃЉрЃБрЃџрЃў)

### рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃў:

1. **рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ PowerShell** (рЃарЃЮрЃњрЃЮрЃарЃф Administrator рЃљрЃа рЃљрЃарЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ)
   - рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ `Win + X`
   - рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ "Windows PowerShell" рЃљрЃю "Terminal"

2. **рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ рЃћрЃА рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ:**
   ```powershell
   winget install Gyan.FFmpeg
   ```

3. **рЃЊрЃљрЃћрЃџрЃЮрЃЊрЃћрЃЌ рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃљрЃА** (1-2 рЃгрЃБрЃЌрЃў)

4. **рЃЊрЃљрЃ«рЃБрЃарЃћрЃЌ рЃЊрЃљ рЃ«рЃћрЃџрЃљрЃ«рЃџрЃљ рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ Terminal** (PATH рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА)

5. **рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ:**
   ```powershell
   ffmpeg -version
   ```

## ­ЪћД рЃЏрЃћрЃЌрЃЮрЃЊрЃў 2: Chocolatey

рЃЌрЃБ рЃњрЃљрЃЦрЃЋрЃЌ Chocolatey рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў:

   - рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ `Win + X`
   - рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ "Windows PowerShell (Admin)"

## ­ЪЊд рЃЏрЃћрЃЌрЃЮрЃЊрЃў 3: Manual Installation (рЃЌрЃБ рЃАрЃ«рЃЋрЃљ рЃљрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА)

### рЃюрЃљрЃЉрЃўрЃ»рЃў 1: рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ

1. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ: **https://www.gyan.dev/ffmpeg/builds/**
2. рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃћрЃЌ: **"ffmpeg-release-essentials.zip"** (рЃљрЃю "ffmpeg-git-essentials.7z")
   - рЃърЃўрЃарЃЊрЃљрЃърЃўрЃарЃў рЃџрЃўрЃюрЃЎрЃў: https://www.gyan.dev/ffmpeg/builds/ffmpeg-release-essentials.zip

### рЃюрЃљрЃЉрЃўрЃ»рЃў 2: рЃњрЃљрЃерЃџрЃљ

1. **рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ** рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃБрЃџрЃў ZIP рЃцрЃљрЃўрЃџрЃў
2. **рЃњрЃљрЃЊрЃљрЃўрЃбрЃљрЃюрЃћрЃЌ** рЃњрЃљрЃ«рЃАрЃюрЃўрЃџрЃў рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћ `C:\` рЃЊрЃўрЃАрЃЎрЃќрЃћ
3. **рЃњрЃљрЃЊрЃљрЃљрЃарЃЦрЃЋрЃўрЃЌ** рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃА: `C:\ffmpeg`

   рЃЊрЃљрЃљрЃ«рЃџрЃЮрЃћрЃЉрЃўрЃЌ рЃљрЃАрЃћ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃБрЃарЃћрЃЉрЃЮрЃЊрЃћрЃА:
   ```
   C:\ffmpeg\
   РћюРћђРћђ bin\
   Рћѓ   РћюРћђРћђ ffmpeg.exe
   Рћѓ   РћюРћђРћђ ffplay.exe
   Рћѓ   РћћРћђРћђ ffprobe.exe
   РћюРћђРћђ doc\
   РћћРћђРћђ presets\
   ```

### рЃюрЃљрЃЉрЃўрЃ»рЃў 3: PATH-рЃерЃў рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ

**рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў A - PowerShell-рЃўрЃЌ (рЃАрЃгрЃарЃљрЃцрЃў):**

1. **рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ PowerShell** (Administrator рЃљрЃа рЃљрЃарЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ)
2. **рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ:**
   ```powershell
   $env:Path += ";C:\ffmpeg\bin"
   [Environment]::SetEnvironmentVariable("Path", $env:Path + ";C:\ffmpeg\bin", "User")
   ```

**рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў B - GUI-рЃЌ:**

1. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ `Win + R`, рЃЕрЃљрЃгрЃћрЃарЃћрЃЌ: `sysdm.cpl` рЃЊрЃљ рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ Enter
2. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ **"Advanced"** рЃбрЃљрЃЉрЃќрЃћ
3. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ **"Environment Variables"**
4. **"User variables"** рЃАрЃћрЃЦрЃфрЃўрЃљрЃерЃў рЃўрЃърЃЮрЃЋрЃћрЃЌ **"Path"**
5. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ **"Edit"**
6. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ **"New"**
7. рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ: `C:\ffmpeg\bin`
8. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ **"OK"** рЃДрЃЋрЃћрЃџрЃљ рЃцрЃљрЃюрЃ»рЃљрЃарЃљрЃќрЃћ

### рЃюрЃљрЃЉрЃўрЃ»рЃў 4: рЃЋрЃћрЃарЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљ

1. **рЃЊрЃљрЃ«рЃБрЃарЃћрЃЌ рЃДрЃЋрЃћрЃџрЃљ Terminal/Command Prompt**
2. **рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃљрЃ«рЃљрЃџрЃў PowerShell рЃљрЃю Command Prompt**
3. **рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ:**
   ```powershell
   ffmpeg -version
   ```

   рЃЌрЃБ рЃЊрЃљрЃўрЃюрЃљрЃ«рЃћрЃЌ рЃЋрЃћрЃарЃАрЃўрЃљ рЃЊрЃљ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ, рЃЏрЃљрЃерЃўрЃю рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃЎрЃљрЃарЃњрЃљрЃЊ рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃљ! РюЁ

## ­Ъћ┤ Troubleshooting

### рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ: winget рЃљрЃа рЃўрЃФрЃћрЃЉрЃюрЃћрЃЉрЃљ

**рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЋрЃљрЃџрЃў:**

## РюЁ рЃарЃЮрЃњрЃЮрЃа рЃЊрЃљрЃЋрЃарЃгрЃЏрЃБрЃюрЃЊрЃћ рЃарЃЮрЃЏ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА?

## ­Ъџђ ffmpeg рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ
